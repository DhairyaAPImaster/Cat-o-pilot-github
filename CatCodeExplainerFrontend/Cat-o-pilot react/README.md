# Cat-o-Pilot Frontend

A minimalistic landing page for the Cat-o-Pilot VS Code extension.

## Getting Started

Simply open `index.html` in your browser to view the website.

## Assets Needed

Place a cat meow sound file at `assets/meow.mp3` for the meow button to work.

You can find free cat meow sounds at:
- freesound.org
- zapsplat.com
- mixkit.co

## Features

- Responsive design
- Cat animations
- Smooth scrolling
- Interactive meow button
- Clean and minimal aesthetic

## Structure

```
├── index.html       # Main HTML file
├── style.css        # All styling
├── script.js        # Interactive elements
└── assets/
    └── meow.mp3     # Cat meow sound (add this)
```
