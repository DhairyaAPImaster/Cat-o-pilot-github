# Troubleshooting

Use this list when the extension is quiet or slow.

## No output

- Confirm a selection is active in the editor.
- Reload the VS Code window.
- Try the command palette action instead of context menu.

## Slow responses

- Reduce the token limit.
- Use shorter selections.

## Install issues

- Make sure the VSIX finished downloading.
- Reinstall the VSIX and reload VS Code.
