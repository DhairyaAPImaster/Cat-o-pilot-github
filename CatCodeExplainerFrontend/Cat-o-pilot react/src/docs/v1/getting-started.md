# Getting Started

Cat-o-Pilot helps you explain code with one prompt. Use this guide to get your first explanation in minutes.

## Install the extension

1. Download a VSIX from the download page.
2. Open VS Code and choose Extensions.
3. Pick **Install from VSIX** in the overflow menu.
4. Reload the window after install.

## Run your first explain

- Open any code file.
- Select a block of code.
- Open the command palette and run **Cat-o-Pilot: Explain Selection**.

## Recommended next steps

- Try a follow up question such as "make it simpler" or "add an example".
- Open the comparison page to see feature differences between versions.
