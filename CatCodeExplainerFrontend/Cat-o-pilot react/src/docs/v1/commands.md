# Commands

The extension adds a few command palette actions. You can also trigger them from the context menu.

## Command palette

- Cat-o-Pilot: Explain Selection
- Cat-o-Pilot: Summarize File
- Cat-o-Pilot: Translate to Plain English

## Context menu

Right click inside the editor and choose **Explain** after selecting code.

## Tips

- Use smaller selections for faster responses.
- If the output is long, ask for a summary instead of rerunning the command.
