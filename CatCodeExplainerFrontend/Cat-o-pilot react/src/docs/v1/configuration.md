# Configuration

Tune the experience by editing your VS Code settings JSON.

## Suggested settings

```json
{
  "catopilot.explainTone": "clear",
  "catopilot.includeExamples": true,
  "catopilot.maxTokens": 800,
  "catopilot.streaming": false
}
```

## What each setting does

- **explainTone**: changes the voice of the response.
- **includeExamples**: adds short examples when relevant.
- **maxTokens**: limits output length.
- **streaming**: shows tokens as they arrive.
