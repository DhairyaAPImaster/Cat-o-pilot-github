# Getting Started (v2)

Version 2 adds faster streaming and deeper context hints. Use this flow to verify everything works.

## Install the build

1. Download the latest VSIX from the download page.
2. Install it through Extensions -> Install from VSIX.
3. Reload VS Code.

## Verify model routing

Run **Cat-o-Pilot: Explain Selection** and confirm the response includes a short "context check" section.

## What is new in v2

- Faster streaming responses.
- Automatic context hints.
- Stronger prompt templates for edge cases.
