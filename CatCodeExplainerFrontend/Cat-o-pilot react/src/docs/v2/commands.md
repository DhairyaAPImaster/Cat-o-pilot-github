# Commands (v2)

Version 2 keeps the core commands and adds a few improvements.

## Command palette

- Cat-o-Pilot: Explain Selection
- Cat-o-Pilot: Summarize File
- Cat-o-Pilot: Translate to Plain English
- Cat-o-Pilot: Explain and Refactor

## Quick tips

- The **Explain and Refactor** command creates a clean rewrite next to the explanation.
- Use **Summarize File** before asking for a refactor on large files.
