# Troubleshooting (v2)

Use these checks to unblock most issues.

## No response appears

- Ensure a selection is active.
- Confirm streaming is enabled.
- Reload VS Code after install.

## Model feels slow

- Lower the token limit.
- Switch to smaller selections.

## Output lacks context

- Set **contextMode** to strict and retry.
- Provide a short prompt like "assume this is part of a payment service".
